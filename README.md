# Legacy Quest Studios 🎮

A cinematic gaming studio landing page inspired by the atmosphere of AAA fantasy games.  
Built with a dark fantasy aesthetic, immersive visuals, and modern UI design.

## 🚀 Features

- Cinematic hero section with fantasy gaming visuals
- Dark themed UI design
- Custom navigation bar
- Game showcase sections
- Responsive layout
- Custom favicon and branding elements
- Smooth visual effects and premium gaming aesthetics

## 🛠️ Technologies Used

- HTML5
- CSS3
- Font Awesome Icons
